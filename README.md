#超市会员管理系统

功能文档

会员模块
1.添加会员信息
 a)存储到member表
 b)数据验证
 c)存储头像图片(存储图片相对路径到member表)
2.查询会员信息
 a)查询member表
3.修改会员状态(正常,挂失,冻结)
 a)存储到member表
4.删除会员
 a)根据会员名存储到member表
5.会员类别
 a)原有会员类别信息修改
 b)添加新的会员类别
 c)存储到member_grade表
6.兑换礼品
 a)查询礼品表gift
 b)如果积分足够,兑换gift表中的礼品
7.充值会员余额
 a)修改member表中的积分字段数据
商品功能 
1.添加商品
 a)商品名，购买后获取积分， 商品数量 ，商品价格
 b)显示商品添加的表单
2.商品修改
 a)分页显示商品的信息 pageSize = 3
 b)点击修改按钮进入修改界面
  i.修改界面：需要有选中商品的信息回显
  ii.商品名称，数量，价格，可得积分
3.商品购买
 a)手动输入 商品名，会员号码
 b)可以选择现金/余额支付
 c)点击购买商品  提交时需要验证  商品号码（在前端查询不到），会员号码 （修改），
   通过会员账号 和 商品名称来 验证 或者加一个 商品查询
  i.非空验证
  ii.数据正确性验证
4.积分记录
 a)分页显示积分记录
 b)模糊查询会员账号

生日提醒
 1.如果有会员的 当天生日 ，可以发送邮件
   需要  邮件系统 

积分抽奖
 输入积分 ，点击随机分配按钮，随机将积分加到 随机用户的 账号上
 按钮也需要 非空验证

登陆系统

1.管理员信息（修改界面）
 a)默认显示当前登陆的 账号信息，
 b)数据正确性/非空 验证
2.添加管理员
 a)添加按钮
 b)重置数据按钮
3.删除管理员
 a)分页显示管理员信息
 b)删除按钮 

礼品消费：
1.添加礼品
 A)礼品名，礼品积分，礼品数量，礼品价值
  非空验证
2.修改礼品数量
3.删除礼品
4.会员兑换礼品的记录
 A)兑换记录id，会员账号，会员名，会员电话，会员邮箱，礼品名称，礼品价值，礼品积分
 B)搜索指定会员的兑换记录
   支持模糊搜索
